# 10장. 미니 프로젝트 완성

### ― 처음으로 ‘내 프로그램’을 만들다

## 1. 이 장이 가장 중요하다

지금까지 우리는

* 입력하고
* 계산하고
* 반복하고
* 판단하고
* 함수로 묶고
* 그림도 그렸다.

이제 마지막 단계다.

> **“이걸 하나로 묶어
> 진짜 프로그램을 만든다.”**

이 장의 목표는 단 하나다.

> **끝까지 완성해 보는 경험**

완벽하지 않아도 된다.
짧아도 된다.
**끝까지 만드는 것**이 핵심이다.

---

## 2. 오늘의 미니 프로젝트

### 🎯 숫자 맞히기 게임 (3번 도전)

게임 규칙은 아주 단순하다.

* 컴퓨터가 숫자 하나를 정한다
* 나는 숫자를 입력한다
* 맞으면 “정답!”
* 틀리면 다시 도전
* 기회는 총 **3번**

👉 지금까지 배운 것들이
👉 **모두 들어간다.**

---

## 3. 먼저 전체 코드를 보자

아래 코드를
**그대로 따라 치세요.**

```python
import random

answer = random.randint(1, 10)

for i in range(3):
    guess = int(input("숫자 맞혀보세요: "))
    
    if guess == answer:
        print("정답!")
        break
    else:
        print("틀림!")

print("게임 끝!")
```

실행해 보고
직접 숫자를 입력해 보자.

---

## 4. 이 코드에 들어 있는 것들

이 짧은 코드 안에는
우리가 배운 핵심이 모두 들어 있다.

* `input()` → 사용자 입력
* `int()` → 숫자로 변환
* `for` → 3번 반복
* `if / else` → 판단
* `break` → 반복 중단

👉 이걸 만들 수 있다면
👉 **당신은 이미 코딩을 하고 있다.**

---

## 5. break는 무슨 역할일까?

```python
if guess == answer:
    print("정답!")
    break
```

이 부분은 이렇게 읽으면 된다.

> “정답이면
> 더 이상 하지 말고
> 바로 끝내라.”

즉,
**게임 종료 버튼**이다.

---

## 6. 이제 하나씩 바꿔 보자

아래 중
하나만 바꿔 보자.

### ① 기회 늘리기

```python
for i in range(5):
```

### ② 범위 바꾸기

```python
answer = random.randint(1, 20)
```

### ③ 메시지 바꾸기

```python
print("아쉽네요!")
```

👉 코드 한 줄이
👉 게임을 바꾼다.

---

## 7. (선택) 정답 알려주기

게임이 끝난 뒤
정답을 보여 주고 싶다면
이 줄을 추가해 보자.

```python
print("정답은", answer)
```

👉 게임이
👉 더 친절해진다.

---

설명할 수 있다면
👉 **이 코드는 이미 당신 것이다.**
