# 📘 GPT와 코딩 대화하기: 파이썬

> **코드를 외우지 말고, 질문하라**  
> GPT와 대화하며 배우는 완전 초보자용 파이썬 입문 교재

---

## 📌 이 저장소는 무엇인가요?

이 저장소는  
**「GPT와 코딩 대화하기: 파이썬」**  
50쪽 분량의 **완전 초보자용 파이썬 교재 원고**를  
Markdown(.md) 형식으로 정리한 자료입니다.

- 코딩을 **처음 접하는 학습자**
- **IDLE 환경**에서 바로 실습하는 구성
- **GPT를 학습 파트너**로 활용하는 대화형 학습 방식

을 핵심 콘셉트로 합니다.

---

## 🎯 이 교재의 특징

- ❌ 문법 암기 중심 아님  
- ✅ **성공 경험 → 질문 → 이해** 흐름
- ✅ 모든 장에 **직접 타이핑 실습**
- ✅ **초보자 에러를 미리 설명**
- ✅ GPT에게 *어떻게 물어볼지*까지 안내

> 이 책은 파이썬 책이면서,  
> 동시에 **GPT 활용 가이드**입니다.

---

## 📂 파일 구성 안내

### 📖 본문 (10장)

| 파일명 | 내용 |
|------|------|
| `01장. 왜 지금 파이썬인가?.md` | 왜 지금 코딩이고, 왜 파이썬인지 |
| `02장. 파이썬 설치와 실행.md` | 설치부터 IDLE 실행까지 |

| `05장. 반복문 for.md` | 반복의 개념과 실습 |

## 🧑‍🎓 이런 분께 추천합니다

- 파이썬을 **처음 설치해 보는 사람**
- 코딩 책을 **끝까지 못 읽어본 경험이 있는 사람**
- 혼자 공부하다가 **에러에서 막혔던 사람**
- GPT를 **학습 도구로 제대로 쓰고 싶은 사람**
- 중·고등학생, 대학 교양, 비전공자 성인 학습자

---

## 🧠 학습 방법 추천

1. 각 장의 코드를 **그대로 따라 치기**
2. 실행 결과를 반드시 확인
3. 장에 있는 **GPT 질문 예시를 그대로 사용**
4. 숫자·조건·문장을 하나씩 바꿔 보기
5. 이해 안 되면 **GPT에게 다시 질문**

> ✔ 이해보다 **손을 먼저 움직이세요**
