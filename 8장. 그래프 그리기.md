## 1. 왜 그래프를 그릴까?

지금까지 우리는
숫자를 입력하고, 계산하고, 글자로 결과를 봤다.

이제 이런 생각이 들 수 있다.

> “한눈에 보고 싶은데요?”
> “숫자가 어떻게 변하는지 알고 싶어요.”

이럴 때 필요한 것이 **그래프**다.

그래프는
👉 숫자를 **그림으로 보여 주는 방법**이다.

---

## 2. 파이썬으로 그림을 그릴 수 있을까?

가능하다.
그리고 생각보다 **아주 쉽다.**

파이썬에서는
**matplotlib**이라는 도구를 사용한다.

이 장의 목표는 이것 하나다.

> **직선을 하나 그려 본다.**

---

## 3. 그래프를 그리기 위한 첫 줄

그래프를 그리려면
먼저 이 한 줄가 필요하다.

```python
import matplotlib.pyplot as plt
```

이 문장은 이렇게 이해하면 된다.

> “그래프 도구를 준비한다.”

---

## 4. 가장 단순한 그래프 (그대로 따라 치세요)

아래 코드를 **그대로 입력**해 보자.

```python
import matplotlib.pyplot as plt

x = range(10)
y = x

plt.plot(x, y)
plt.show()
```

실행하면
직선 하나가 화면에 나타난다.

🎉
**숫자가 그림이 되었다.**

---

## 5. 이 코드의 의미를 아주 쉽게 보면

```python
x = range(10)
```

→ 가로축 값 (0부터 9)

```python
y = x
```

→ 세로축 값 (x와 같다)

```python
plt.plot(x, y)
```

→ 점들을 선으로 연결한다

```python
plt.show()
```

→ 화면에 보여 준다

---

## 6. 원점을 지나는 45도 직선

방금 그린 그래프는
수학적으로 이렇게 쓸 수 있다.

> **y = x**

그래서
원점을 지나고
기울기 1인 **45도 직선**이 나온다.

만약
비율이 어색해 보인다면
이 한 줄을 추가해 보자.

```python
plt.axis('equal')
```

---

## 7. 기울기를 바꿔 보자

딱 한 줄만 바꿔 보자.

```python
y = [i * 2 for i in x]
```

전체 코드는 이렇게 된다.

```python
import matplotlib.pyplot as plt

x = range(10)
y = [i * 2 for i in x]

plt.plot(x, y)
plt.show()
```

👉 기울기가 더 가파른
👉 새로운 직선이 그려진다.

---

## 8. 수평선도 그릴 수 있다

이번에는
값이 변하지 않는 선을 그려 보자.

```python
x = range(10)
y = [5] * 10
```

이 코드는
**y = 5** 인 수평선을 만든다.

👉 리스트에 같은 값만 있으면
👉 수평 직선이 된다.

---

## 9. 그래프에서 꼭 기억할 5줄 공식

그래프는
아래 **5줄 패턴**으로 거의 끝난다.

```python
import matplotlib.pyplot as plt
x = range(10)
y = x
plt.plot(x, y)
plt.show()
```

> **그래프는
> y 한 줄만 바꾸는 놀이**다.
