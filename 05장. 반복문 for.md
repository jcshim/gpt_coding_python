# 5장. 반복문 for

### ― 컴퓨터를 지치지 않는 일꾼으로 만들다

## 1. 같은 일을 여러 번 시키고 싶을 때

지금까지 우리는
한 줄 한 줄 직접 일을 시켰다.

```python
print("Hello")
print("Hello")
print("Hello")
```

이렇게도 되긴 한다.
하지만 곧 이런 생각이 든다.

> “이걸 10번, 100번 하려면
> 계속 써야 하나요?”

여기서 등장하는 것이
**반복문(for)** 이다.

---

## 2. for 문은 이렇게 생겼다

파이썬에서 가장 기본적인 반복문은 이것이다.

```python
for i in range(5):
    print(i)
```

이 코드는 이렇게 읽으면 된다.

> “i가 0부터 4까지 바뀌면서
> 아래 일을 반복해라.”

---

## 3. 직접 실행해 보자 (그대로 따라 치세요)

```python
for i in range(5):
    print(i)
```

실행 결과는 이렇게 나온다.

```
0
1
2
3
4
```

👉 **총 5번 실행**되었다.

---

## 4. 왜 1부터가 아니라 0부터일까?

많은 초보자가
여기서 바로 묻는다.

> “왜 1이 아니라 0부터예요?”

지금은 이렇게만 받아들이자.

> **컴퓨터는 0부터 세는 걸 좋아한다.**

파이썬도 그렇다.

---

## 5. 1부터 5까지 출력하려면?

조금만 바꿔 보자.

```python
for i in range(1, 6):
    print(i)
```

출력은 이렇게 된다.

```
1
2
3
4
5
```

### 핵심 규칙 하나

> **range(시작, 끝)** 에서
> **끝 숫자는 포함되지 않는다.**

---

## 6. 반복문에서 꼭 지켜야 할 규칙

아래 코드를 보자.

```python
for i in range(5):
print(i)
```

이 코드는 **에러**가 난다.

왜일까?

👉 **들여쓰기(Indentation)** 가 없기 때문이다.

---

### 올바른 형태

```python
for i in range(5):
    print(i)
```

파이썬에서는
들여쓰기가 **규칙**이다.

> **반복해서 할 일은
> 반드시 안쪽으로 들여쓴다.**

---

## 7. 반복문으로 별(*) 찍기

이제 재미있는 걸 해 보자.

```python
for i in range(1, 6):
    print("*" * i)
```

출력 결과:

```
*
**
***
****
*****
```

👉 눈으로 바로 결과가 보인다.
👉 **반복의 힘을 느끼는 순간**이다.

---

## 8. 반복문과 계산을 함께 쓰면?

```python
for i in range(1, 6):
    print(i * 2)
```

출력:

```
2
4
6
8
10
```

👉 같은 계산을
👉 여러 번 자동으로 한다.

---

## 9. 반복문의 핵심은 이것 하나

반복문을 외울 필요는 없다.
이 문장 하나면 충분하다.

> **for = 같은 일을 여러 번 시킨다**

---

## 10. GPT에게 이렇게 질문해 보자

이 장을 마쳤다면
GPT에게 이렇게 물어보자.

> “for 문을
> 초등학생에게 설명하듯 알려줘.”

또는

> “range(1, 6)이
> 왜 1부터 5까지인지
> 그림으로 설명해 줘.”

GPT는
**반복 개념을 이해시키는 설명가**가 되어 준다.

---

## 11. 이 장의 핵심 한 문장

> **for 문은
> 컴퓨터를 지치지 않는 일꾼으로 만든다.**

---

다음 장에서는
👉 **조건에 따라 다른 일을 시키는 방법**,
즉 **if 문**을 배운다.

이제
컴퓨터가 **생각하는 척하기 시작**한다.
