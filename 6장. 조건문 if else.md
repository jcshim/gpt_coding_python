# 6장. 조건문 if / else

### ― 컴퓨터가 선택하게 하다

## 1. 이제 컴퓨터에게 ‘판단’을 맡겨 보자

앞 장에서 우리는
컴퓨터에게 **같은 일을 여러 번** 시켰다.

이제는 이런 요구가 생긴다.

> “조건에 따라
> 다른 일을 하게 하고 싶어요.”

예를 들어,

* 숫자가 크면 “합격”
* 작으면 “불합격”
* 맞으면 “정답”
* 틀리면 “다시 도전”

이럴 때 사용하는 것이
**조건문(if / else)** 이다.

---

## 2. if 문은 이렇게 생겼다

가장 기본적인 형태는 이것이다.

```python
if 조건:
    실행할 일
```

이 코드는 이렇게 읽으면 된다.

> “만약 조건이 맞으면
> 안에 있는 일을 해라.”

---

## 3. 직접 실행해 보자 (그대로 따라 치세요)

```python
n = int(input())

if n > 10:
    print("10보다 큽니다")
```

예를 들어 `15`를 입력하면

```
10보다 큽니다
```

`5`를 입력하면
아무것도 나오지 않는다.

👉 조건이 **거짓**이면
아무 일도 하지 않는다.

---

## 4. else는 ‘그렇지 않으면’이다

조건이 맞지 않을 때
다른 일을 시키고 싶다면
`else`를 사용한다.

```python
n = int(input())

if n > 10:
    print("10보다 큽니다")
else:
    print("10 이하입니다")
```

👉 이제
어떤 경우든 결과가 나온다.

---

## 5. if / else는 이렇게 기억하자

> **if = 만약
> else = 그렇지 않으면**

이 두 단어만 기억하면 된다.

---

## 6. 가장 많이 쓰는 예제: 짝수 / 홀수

```python
n = int(input())

if n % 2 == 0:
    print("짝수")
else:
    print("홀수")
```

여기서 새로운 기호가 등장했다.

* `%` → 나머지
* `==` → 같은지 비교

👉 `=` 와 `==` 는 **다르다.**

---

## 7. 초보자가 가장 많이 틀리는 부분

### ❌ 잘못된 예

```python
if n = 10:
    print("같다")
```

이건 **에러**다.

---

### ✅ 올바른 비교

```python
if n == 10:
    print("같다")
```

> **=`는 저장  
> `==`는 비교**

이 차이는 매우 중요하다.

---

## 8. 조건문으로 게임을 만들어 보자

이제 재미있는 예제를 보자.

```python
answer = 7
guess = int(input("숫자를 맞혀보세요: "))

if guess == answer:
    print("정답!")
else:
    print("틀림!")
```

👉 컴퓨터가
👉 **맞았는지 틀렸는지 판단**한다.

이 순간부터
코딩은 **게임이 된다.**

---

## 9. 조건은 얼마든지 바꿀 수 있다

아래 중 하나만 바꿔 보자.

1️⃣ `==` 를 `>` 로 바꾸기
2️⃣ 정답 숫자를 바꾸기
3️⃣ 메시지를 바꾸기

👉 결과가 어떻게 달라지는지 보자.

---

## 10. GPT에게 이렇게 질문해 보자

이 장을 마쳤다면
GPT에게 이렇게 물어보자.

> “if / else를
> 초등학생에게 설명하듯 알려줘.”

또는

> “=`와 `==`의 차이를
> 예제로 설명해 줘.”

GPT는
**판단 구조를 설명해 주는 선생님**이다.

---

## 11. 이 장의 핵심 한 문장

> **if / else는
> 컴퓨터에게 선택권을 주는 문법이다.**

---

다음 장에서는
👉 **코드에 이름을 붙여 재사용하는 방법**,
즉 **함수(function)** 를 배운다.

이제
컴퓨터가 **기억하며 일하기 시작**한다.
