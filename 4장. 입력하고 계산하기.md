# 4장. 입력하고 계산하기

### ― 컴퓨터가 내 말을 듣기 시작한다

## 1. 이제 컴퓨터에게 말을 걸어 보자

앞 장까지 우리는
컴퓨터에게 **정해진 일**만 시켰다.

```python
a = 10
b = 20
```

이제 한 단계 더 나아가 보자.

> **“내가 직접 숫자를 넣고 싶어요.”**

이 순간부터
코딩은 **진짜 대화**가 된다.

---

## 2. 컴퓨터에게 입력을 받는 방법

파이썬에서는
사용자의 입력을 이렇게 받는다.

```python
input()
```

이 한 줄은 컴퓨터가 이렇게 말하는 것이다.

> “기다릴게요.
> 당신이 키보드로 입력하세요.”

---

## 3. 첫 입력 실습 (그대로 따라 치세요)

아래 코드를 그대로 입력해 보자.

```python
x = input()
print(x)
```

실행 후
예를 들어 `10`을 입력하면
화면에 다시 `10`이 출력된다.

👉 컴퓨터가
👉 **당신의 말을 들었다.**

---

## 4. 그런데 이상한 일이 생긴다

이제 두 수를 더해 보자.

```python
a = input()
b = input()
print(a + b)
```

`10`과 `20`을 입력하면
결과는 이렇게 나온다.

```
1020
```

> “어? 30이 아닌데요?”

이건 **에러가 아니다.**
아주 중요한 개념 때문이다.

---

## 5. 입력은 ‘글자’로 들어온다

파이썬에서 `input()`으로 받은 값은
모두 **문자(글자)**다.

즉,

* `"10"`
* `"20"`

이렇게 들어온다.

그래서

```python
"10" + "20"
```

은
👉 `"1020"` 이 된다.

---

## 6. 숫자로 바꿔 주는 주문: int()

문자를 숫자로 바꾸려면
이 주문을 사용한다.

```python
int()
```

이제 코드를 이렇게 바꿔 보자.

```python
a = int(input())
b = int(input())
print(a + b)
```

`10`과 `20`을 입력하면
결과는 이렇게 나온다.

```
30
```

🎉 드디어
**진짜 계산**이 되었다.

---

## 7. 이 구조를 외울 필요는 없다

다만
이 흐름은 꼭 기억하자.

> **입력 → 변환 → 계산 → 출력**

```python
a = int(input())
b = int(input())
print(a + b)
```

이 패턴은
앞으로 수없이 다시 등장한다.

---

## 8. 이제 조금만 바꿔 보자

아래 중 하나만 바꿔 보자.

1️⃣ `+` 를 `*` 로 바꾸기
2️⃣ `print(a + b)` 를 `print(a - b)` 로 바꾸기
3️⃣ 숫자를 직접 입력해 보기

👉 결과가 어떻게 달라지는지 관찰해 보자.

---

## 9. 초보자가 가장 많이 하는 실수

### ❌ 실수 1

```python
a = input()
b = input()
print(int(a + b))
```

이건
`"10" + "20"` 을 먼저 하고
그다음에 숫자로 바꾸려는 코드다.

👉 **순서가 중요하다.**

---

### ✅ 올바른 순서

```python
a = int(input())
b = int(input())
print(a + b)
```

---

## 10. GPT에게 이렇게 질문해 보자

이 장을 마쳤다면
GPT에게 이렇게 물어보자.

> 아주 쉽게 설명해 주세요.”

또는

> “int(input()) 이 구조를
> 그림으로 설명해 주세요.”

GPT는
**이해를 돕는 설명가**가 되어 준다.

---

## 11. 이 장의 핵심 한 문장

> **컴퓨터는
> 내가 입력한 걸
> 먼저 ‘글자’로 듣는다.**
