# 7장. 함수(function)

### ― 코드에 이름을 붙이다

## 1. 같은 코드를 또 쓰고 싶을 때

앞 장까지 오면서
이런 생각이 들었을 것이다.

> “이 계산,
> 다음에도 또 쓰고 싶은데요?”

지금까지는
필요할 때마다
같은 코드를 다시 써야 했다.

```python
a = 3
b = 4
print(a + b)
```

이걸
다른 곳에서도 쓰고 싶다면
어떻게 해야 할까?

👉 **코드에 이름을 붙이면 된다.**

그게 바로 **함수(function)** 다.

---

## 2. 함수는 이렇게 생각하자

함수를 한 문장으로 말하면 이렇다.

> **함수는 ‘이름 붙인 코드 묶음’이다.**

* 자주 쓰는 코드에 이름을 붙이고
* 필요할 때 그 이름을 부른다

---

## 3. 가장 쉬운 함수 하나 만들어 보기

아래 코드를 **그대로 따라 치세요.**

```python
def add(a, b):
    return a + b
```

입력하고 나서
**반드시 Enter를 한 번 더 눌러야 한다.**

IDLE에서는
이 빈 줄이 **함수 정의의 끝**이다.

---

## 4. 함수 사용해 보기

이제 방금 만든 함수를 사용해 보자.

```python
print(add(3, 4))
```

출력:

```
7
```

🎉
우리는 방금
**새로운 ‘기능’을 하나 만든 것**이다.

---

## 5. 함수의 구조를 그림처럼 보자

```python
def add(a, b):
    return a + b
```

이 구조는 이렇게 이해하면 된다.

* `def` → “함수 만들겠다”
* `add` → 함수 이름
* `(a, b)` → 재료
* `return` → 결과를 돌려준다

👉 **입력 → 처리 → 결과**

---

## 6. IDLE에서 초보자가 가장 많이 하는 실수

IDLE에서
아래처럼 치면 에러가 난다.

```python
def add(a, b):
    return a + b
print(add(3, 4))
```

왜일까?

👉 **함수가 아직 끝나지 않았다고 생각하기 때문**이다.

---

### ✅ 올바른 방법 (중요!)

```python
def add(a, b):
    return a + b

print(add(3, 4))
```

> **함수 입력 후
> 빈 줄(Enter)을 꼭 한 번 더!**

---

## 7. 함수는 왜 쓰는 걸까?

함수의 장점은 딱 세 가지다.

1️⃣ **코드를 줄인다**
2️⃣ **읽기 쉬워진다**
3️⃣ **다시 쓸 수 있다**

예를 들어,

```python
print(add(10, 20))
print(add(5, 7))
print(add(100, 3))
```

같은 계산을
여러 번 쉽게 할 수 있다.

---

## 8. 함수 이름은 이렇게 짓자

함수 이름은
**하는 일을 말해 주면 된다.**

* `add` → 더하기
* `mul` → 곱하기
* `check_even` → 짝수 확인

👉 영어를 잘 몰라도
의미만 통하면 된다.

---

## 9. 함수와 지금까지 배운 것 연결하기

함수 안에는
우리가 배운 모든 것이 들어갈 수 있다.

```python
def double_input():
    n = int(input())
    return n * 2
```

```python
print(double_input())
```

👉 입력
👉 계산
👉 결과 반환

모두 가능하다.
