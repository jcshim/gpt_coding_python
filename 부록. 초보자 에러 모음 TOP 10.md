# 📘 부록. 초보자 에러 모음 TOP 10

### ― 에러는 실패가 아니라 힌트다

> **에러가 난다는 건,
> 지금 제대로 배우고 있다는 뜻이다.**

초보자가 가장 자주 만나는 에러 10가지를
**원인 → 해결 → GPT 질문**으로 정리했다.

---

## 1️⃣ `SyntaxError: invalid syntax`

### ❓ 왜 나왔을까?

* 괄호 `()` 빠짐
* 따옴표 `" "` 빠짐
* `:`(콜론) 빠짐

### ❌ 예

```python
if a > 10
    print(a)
```

### ✅ 해결

```python
if a > 10:
    print(a)
```

### 🤖 GPT에게 이렇게 물어보자

> “이 SyntaxError가
> 어느 줄에서 왜 났는지
> 초보자 기준으로 설명해 줘.”

---

## 2️⃣ `IndentationError: expected an indented block`

### ❓ 왜 나왔을까?

* `if`, `for`, `def` 다음 줄에
  **들여쓰기 안 함**

### ❌ 예

```python
for i in range(3):
print(i)
```

### ✅ 해결

```python
for i in range(3):
    print(i)
```

### 🧠 한 줄 기억

> **파이썬은 들여쓰기가 규칙이다.**

---

## 3️⃣ `NameError: name 'x' is not defined`

### ❓ 왜 나왔을까?

* 변수를 만들기 전에 사용함
* 이름을 잘못 씀

### ❌ 예

```python
print(a)
```

### ✅ 해결

```python
a = 10
print(a)
```

### 🤖 GPT 질문

> “이 NameError는
> 어떤 이름을 못 찾았다는 뜻인가요?”

---

## 4️⃣ `TypeError`

### ❓ 왜 나왔을까?

* 글자와 숫자를 섞어서 계산함

### ❌ 예

```python
a = input()
print(a + 10)
```

### ✅ 해결

```python
a = int(input())
print(a + 10)
```

### 🧠 핵심

> **input()은 항상 글자다.**

---

## 5️⃣ 결과가 안 나오는데, 에러도 없다

### ❓ 왜 그럴까?

* `print()`를 안 씀

### ❌ 예

```python
a = 10 + 20
```

### ✅ 해결

```python
print(a)
```

### 🧠 한 줄 기억

> **보여 주려면 print()**

---

## 6️⃣ `=` 와 `==` 헷갈림

### ❓ 왜 나왔을까?

* 비교해야 하는데 저장함

### ❌ 예

```python
if a = 10:
```

### ✅ 해결

```python
if a == 10:
```

### 🧠 기억법

> **= 는 저장
> == 는 비교**

---

## 7️⃣ 함수가 있다고 생각했는데 없다고 나옴

### 에러

```text
NameError: name 'add' is not defined
```

### ❓ 왜 나왔을까?

* 함수 정의 중 에러 발생
* IDLE에서 **빈 줄(Enter)** 안 누름

### ✅ 해결

```python
def add(a, b):
    return a + b

print(add(3, 4))
```

### 🧠 중요

> **def 뒤에는
> Enter를 한 번 더!**

---

## 8️⃣ `range()`가 생각과 다르게 동작함

### ❓ 왜 1~5가 안 나올까?

```python
for i in range(5):
    print(i)
```

### 출력

```
0 1 2 3 4
```

### ✅ 해결

```python
for i in range(1, 6):
    print(i)
```

### 🧠 규칙

> **끝 숫자는 포함되지 않는다.**

---

## 9️⃣ 그래프가 안 나옴

### ❓ 왜 안 보일까?

* `plt.show()`를 안 씀

### ❌ 예

```python
plt.plot(x, y)
```

### ✅ 해결

```python
plt.plot(x, y)
plt.show()
```

### 🧠 한 줄

> **그래프는 show()까지가 완성**

---

## 🔟 코드를 복사했는데 안 됨

### ❓ 왜 그럴까?

* 보이지 않는 공백
* 들여쓰기 깨짐
* 특수문자 섞임

### ✅ 해결 방법

* **직접 다시 타이핑**
* 한 줄씩 실행

### 🧠 진실

> **손으로 친 코드만
> 내 실력이 된다.**
